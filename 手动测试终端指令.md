# Agent Service 手动测试终端指令

## 🚀 快速启动服务（在一个终端执行）

```bash
# 方式1：直接使用虚拟环境的 python（推荐）
cd agent_service
../.venv/bin/python main.py serve

# 方式2：激活虚拟环境后使用
source .venv/bin/activate
cd agent_service
python main.py serve
```

服务会自动在 8000 端口启动（如果被占用会自动找下一个可用端口）

**注意：**
- `main.py` 会自动加载 `.env.agent` 环境变量
- 如果遇到 `python: command not found`，使用 `.venv/bin/python` 或先激活虚拟环境

## 🧪 单条测试（CLI 模式，不启动服务）

```bash
# 方式1：直接使用虚拟环境的 python（推荐）
cd agent_service
../.venv/bin/python main.py chat "北京天气怎么样"
../.venv/bin/python main.py chat "附近的咖啡馆"
../.venv/bin/python main.py chat "三里屯附近的日料"

# 方式2：激活虚拟环境后使用
source .venv/bin/activate
cd agent_service
python main.py chat "北京天气怎么样"
```

**原理：**
- `chat` 命令：单次执行，直接返回 JSON 结果，不启动服务
- `serve` 命令：启动 HTTP 服务，持续运行，接受 API 请求

## 🧪 手动测试命令（HTTP 服务模式，在另一个终端执行）

### 1. 健康检查
```bash
curl http://127.0.0.1:8000/health
```

### 2. 基础对话测试
```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H 'Content-Type: application/json' \
  -d '{"query":"北京天气怎么样"}'
```

### 3. 地点查询测试
```bash
# 附近搜索
curl -X POST http://127.0.0.1:8000/chat \
  -H 'Content-Type: application/json' \
  -d '{"query":"附近的咖啡馆"}'

# 指定地点搜索
curl -X POST http://127.0.0.1:8000/chat \
  -H 'Content-Type: application/json' \
  -d '{"query":"三里屯附近的日料"}'
```

### 4. 多轮对话测试（带 session_id）
```bash
# 第一轮
curl -X POST http://127.0.0.1:8000/chat \
  -H 'Content-Type: application/json' \
  -d '{"query":"郑州天气怎么样", "session_id":"test-001"}'

# 第二轮（指代消解）
curl -X POST http://127.0.0.1:8000/chat \
  -H 'Content-Type: application/json' \
  -d '{"query":"那边有什么好吃的", "session_id":"test-001"}'
```

### 5. Web 搜索测试
```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H 'Content-Type: application/json' \
  -d '{"query":"特斯拉最新股价"}'
```

### 6. 百科查询测试
```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H 'Content-Type: application/json' \
  -d '{"query":"什么是量子计算"}'
```

## 💡 提示

1. **确保 LM Studio 已启动**：服务依赖本地 LM Studio（默认 localhost:1234）
2. **API Keys 已配置**：`.env.agent` 中已包含必要的 API keys
3. **端口自动选择**：如果 8000 被占用，服务会自动尝试 8001-8009
4. **保持终端运行**：启动服务的终端需要保持运行状态
5. **使用 jq 格式化**：安装 jq 可以更好地查看 JSON 响应（`brew install jq`）
