## 生产部署建议

### 架构设计

```
用户 query
    ↓
规则层（拦截极短查询 ≤2字）
    ↓
4B LLM（处理标准 query）
    ↓
置信度检查（< 0.7 触发澄清）
    ↓
Parse 失败兜底（web_search）
```

### 置信度阈值

| 阈值 | 澄清率 | 准确率 | 建议 |
|------|--------|--------|------|
| 0.7 | ~30% | 79% | 平衡 |
| 0.8 | ~40% | 85%+ | 保守 |
| 0.6 | ~20% | 75% | 激进 |

**推荐**：0.7（平衡准确率和用户体验）

### 规则层前置

```python
def pre_filter(query: str) -> Optional[str]:
    # 极短查询（≤2字）直接澄清
    if len(query.strip()) <= 2:
        return "clarify"
    
    # 单字地名 → web_search
    if query in ["北京", "上海", "广州", "深圳", ...]:
        return "web_search"
    
    return None  # 交给 4B
```
