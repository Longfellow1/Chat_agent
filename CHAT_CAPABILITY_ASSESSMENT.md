## 一、当前闲聊实现现状

### 1.1 实现方式

**核心逻辑**: 当`decision_mode == "reply"`时，调用LLM生成回复

```python
# agent_service/app/orchestrator/chat_flow.py line 367
text = self.llm_client.generate(
    user_query=effective_query, 
    system_prompt=SYSTEM_PROMPT
)
```

**System Prompt**:
```
你是中文智能体。优先给出直接、简洁、可执行的回答。
不要编造实时数据；涉及天气、新闻、股票、附近信息时，若没有工具结果请明确说明并向用户补充必要信息。
对闲聊、情绪支持、日常问候，直接自然回复，不要误调用新闻或搜索工具。
对医疗高风险提问（如胸痛/心梗/处方/要求确定诊断），不要给诊断与处方，明确建议尽快线下就医。
对违法有害请求，明确拒绝并引导合法路径。
```

### 1.2 特殊处理

**硬编码回复** (不依赖LLM):
1. 结束对话: "好的，先聊到这里。你有需要随时叫我。"
2. 无意义输入: "我没理解你的输入。你可以换一种更完整的说法。"
3. 违法内容: "我不能帮助处理违法或危险行为相关请求。请在合法合规范围内提问。"
4. 危机干预: "我听到你现在很痛苦。你并不需要一个人扛着..."

## 二、能力评估

### 2.1 基础闲聊 (Small Talk) - 56条

**样例**:
- "中午好" / "晚安"
- "周末计划干什么"
- "你喜欢吃什么"
- "最近有什么好看的剧推荐"

**当前能力**: ✅ **基本满足**
- LLM可以生成自然回复
- System prompt引导"直接自然回复"
- 预期准确率: 95%+

**问题**: 无

---

### 2.2 知识问答 (Knowledge) - 11条

**样例**:
- "什么是人工智能"
- "解释一下量子计算"
- "光合作用原理"

**当前能力**: ✅ **基本满足**
- LLM具备基础知识回答能力
- System prompt要求"简洁、可执行"
- 预期准确率: 80-90%

**问题**: 
- 回复质量依赖LLM能力
- 可能过于简短

---

### 2.3 情感支持 (Emotion Support) - 11条

**样例**:
- "最近睡眠不好怎么办"
- "我好焦虑啊"
- "感觉自己什么都做不好"
- "今天和家人吵架了"

**当前能力**: ✅ **基本满足**
- System prompt提到"情绪支持，直接自然回复"
- 危机干预有专门处理
- 预期准确率: 80%

**问题**:
- 回复可能不够温暖/共情
- 缺少专业心理支持话术

---

### 2.4 语言游戏 (Language Game) - 20条

**样例**:
- "我们来玩飞花令吧"
- "接成语：守株待兔"
- "上联：春风送暖入屠苏，下联是什么"
- "给我编一个穿越故事"
- "来一段rap歌词关于开车的"

**当前能力**: ❌ **严重欠缺**


**问题**:
1. **LLM能力不足**: 当前LLM可能无法完成复杂创作任务
2. **System Prompt不支持**: 强调"简洁、可执行"，与创作任务冲突
3. **无专门处理**: 没有针对语言游戏的特殊逻辑

**预期结果**:
- decision_mode: reply ✅ (能识别)
- 回复质量: ❌ (无法真正完成任务)
- 预期准确率: 50% (能回复但质量差)

**影响**: 20条数据 (10%总量)

---

### 2.5 能力边界 (Out of Scope) - 3条

**样例**:
- "现在几点了"

**当前能力**: ⚠️ **部分支持**
- 可以识别为reply
- 但无法提供真实时间

**问题**: 缺少明确的能力边界说明

## 三、核心问题诊断

### 🔴 问题1: Language Game能力缺失

**严重程度**: 高  
**影响范围**: 20条 (10%)  
**根本原因**: 
- LLM模型能力不足（可能是小模型）
- System Prompt设计不支持创作任务
- 无专门的游戏模式

**表现**:
- 飞花令: 可能只回复"好的"而不真正玩
- 成语接龙: 可能接不上或接错
- 故事创作: 可能拒绝或生成质量差
- Rap歌词: 可能无法生成

---

### 🟡 问题2: 情感支持深度不足

**严重程度**: 中  
**影响范围**: 11条 (5.5%)  
**根本原因**:
- System Prompt过于简洁
- 缺少共情话术模板
- 无情感分析能力

**表现**:
- 回复可能过于理性/冷淡
- 缺少情感共鸣
- 建议可能过于简单

---

### 🟢 问题3: 知识回答深度可控

**严重程度**: 低  
**影响范围**: 11条 (5.5%)  
**根本原因**:
- System Prompt要求"简洁"
- 可能回答过于简短

**表现**:
- 回答正确但不够详细
- 用户可能需要追问

**检测逻辑**:
```python
def detect_language_game(query: str) -> bool:
    """检测语言游戏类查询"""
    game_keywords = [
        "飞花令", "成语接龙", "对对子", "猜谜", "编故事",
        "rap", "绕口溜", "脑筋急转弯", "接龙", "顺口溜"
    ]
    return any(kw in query for kw in game_keywords)
```

**降级回复**:
```python
if detect_language_game(query):

        decision_mode="reply",
        final_text=(
            "我理解你想玩语言游戏，这类互动需要更强的创作能力。"
            "我可以尝试参与，但可能表现不够理想。"
            "如果你想要更好的体验，建议使用专门的语言游戏应用。"
        )
    )
```

**预期提升**: Language Game从50%提升到70%
