## 技术架构

### 核心组件

```
Agent Service
├── Intent Router          # 意图路由（96% 准确率）
├── Query Preprocessor     # Query 预处理
├── Tool Executor          # 工具执行器
│   ├── find_nearby       # 本地生活（规则引擎）
│   ├── web_search        # 联网搜索（Provider Chain）
│   ├── get_weather       # 天气查询
│   ├── get_news          # 新闻查询
│   ├── get_stock         # 股票查询
│   └── plan_trip         # 行程规划（开发中）
├── Result Processor       # 结果处理
│   ├── Rerank            # 重排序
│   ├── Filter            # 过滤
│   └── Template          # 模板化
└── Content Rewriter       # 内容改写
```

### Provider Chain

```
web_search:
  百度 Web 搜索 (优先级 1)
    ↓ fallback
  Bing MCP (优先级 2)
    ↓ fallback
  Tavily (优先级 3)
```
