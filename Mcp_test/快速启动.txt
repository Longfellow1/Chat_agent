🚀 快速启动命令
========================================

1. 确保 OpenAI API Key 已配置（必需）
   编辑 .env 文件，填入 OPENAI_API_KEY

2. 启动项目
   npm run dev

3. 访问前端
   http://localhost:5173

4. 测试高德地图 MCP
   输入：想去北京玩3天，喜欢历史文化和美食
