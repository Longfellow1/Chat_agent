# 🚀 快速开始 - 从这里开始！

## ✅ 环境准备完成

✅ Node.js 环境就绪  
✅ 所有依赖已安装  
✅ 项目结构已创建  
✅ `.env` 文件已创建

## ⚡ 立即开始（只需2步）

### 步骤 1: 配置 OpenAI API Key

编辑 `.env` 文件，填入你的 OpenAI API Key：

```bash
OPENAI_API_KEY=sk-your-actual-api-key-here
```

**获取 API Key:**
1. 访问 https://platform.openai.com/api-keys
2. 登录/注册账号
3. 创建新的 API Key
4. 复制到 `.env` 文件中

### 步骤 2: 启动项目

```bash
npm run dev
```

这将同时启动：
- 后端服务器: http://localhost:3001
- 前端应用: http://localhost:5173

## 🎯 开始测试

1. 打开浏览器访问 **http://localhost:5173**

2. 在表单中输入行程需求，例如：
   ```
   想去北京玩3天，喜欢历史文化和美食
   ```

3. 点击"开始规划行程"

4. 等待 AI 生成行程规划（可能需要几秒钟）

## 📋 关于 MCP 服务

**重要提示：MCP 服务是可选的！**

- ✅ **即使不配置高德地图和小红书 MCP，项目也能正常运行**
- ✅ Agent 会以仅 LLM 模式运行，仍能生成行程建议
- ✅ 配置 MCP 服务可以获得更准确的数据，但不是必需的

如果看到这些警告信息，这是**正常的**：
```
⚠️  高德地图 MCP 客户端连接失败，将在演示模式下运行
⚠️  小红书 MCP 客户端连接失败，将在演示模式下运行
```
