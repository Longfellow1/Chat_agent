## 📦 部署清单

### 必需步骤
1. ✅ 运行部署脚本：`./deploy.sh`
2. ✅ 配置环境变量：编辑 `.env` 文件
3. ✅ 启动服务：`npm run start`（生产）或 `npm run dev`（开发）

### 环境变量配置
**必需配置**:
- `OPENAI_API_KEY` - OpenAI API 密钥（必需）

**可选配置**:
- `AMAP_API_KEY` - 高德地图 API 密钥
- `AMAP_MCP_SERVER_URL` - 高德地图 MCP 服务器地址
- `XHS_MCP_COMMAND` - 小红书 MCP 命令
- `PORT` - 服务器端口（默认 3001）
- `NODE_ENV` - 环境模式（development/production）

---

## 🚀 部署方式

### 方式一：使用一键部署脚本（推荐）

```bash
# 1. 给脚本添加执行权限（如未添加）
chmod +x deploy.sh

# 2. 运行部署脚本
./deploy.sh

# 3. 编辑 .env 文件，填入 API Key

# 4. 启动服务
npm run start  # 生产环境
# 或
npm run dev    # 开发环境
```

### 方式二：手动部署

```bash
# 1. 安装依赖
npm run install:all

# 2. 构建前端
npm run build

# 3. 配置环境变量
cp env.example .env
# 编辑 .env 文件

# 4. 启动服务
npm run start
```

## 📝 给同事的使用说明

### 快速开始（3步）

1. **运行部署脚本**
   ```bash
   ./deploy.sh
   ```

2. **配置 API Key**
   - 编辑 `.env` 文件
   - 填入 `OPENAI_API_KEY`

3. **启动项目**
   ```bash
   npm run dev    # 开发模式（推荐）
   # 或
   npm run start  # 生产模式
   ```

### 访问应用
- **开发模式**: http://localhost:5173（前端）+ http://localhost:3001（后端）
- **生产模式**: http://localhost:3001（前后端统一）
