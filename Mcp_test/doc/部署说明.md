# 🚀 快速部署指南

## 一键部署（推荐）

```bash
# 1. 运行部署脚本
./deploy.sh

# 2. 编辑 .env 文件，填入 OPENAI_API_KEY
# 打开 .env 文件，将 OPENAI_API_KEY 替换为你的实际密钥

# 3. 启动项目
npm run dev    # 开发模式（推荐，前后端分离）
# 或
npm run start  # 生产模式（前后端统一）
```

## 访问应用

- **开发模式**: 打开浏览器访问 http://localhost:5173
- **生产模式**: 打开浏览器访问 http://localhost:3001

## 必需配置

编辑 `.env` 文件，至少需要配置：

```env
OPENAI_API_KEY=sk-your-actual-api-key-here
```

## 系统要求

- Node.js 18 或更高版本
- npm 或 yarn

## 遇到问题？

1. 检查 Node.js 版本：`node --version`（需要 18+）
2. 检查端口是否被占用：`lsof -ti:3001` 或 `lsof -ti:5173`
3. 查看详细文档：`DEPLOYMENT_READINESS.md`
